# 🎙️ Sesle Yazma (Speech to Text) - CachyOS & Arch Linux

Linux sistemlerinde (özellikle CachyOS ve Arch tabanlı dağıtımlarda) sistem genelinde çalışan, stabil ve hızlı bir sesle yazdırma çözümüdür.

---

## 🇹🇷 Türkçe Açıklama

Bu proje, fısıltıları veya ortam seslerini yanlışlıkla sistem komutu (yeni sekme açma vb.) olarak algılamayan, `--clearmodifiers` argümanıyla güçlendirilmiş güvenli bir sesle yazma kurulumu sunar.

### 🚀 Hızlı Kurulum
Terminalinizi açın ve şu komutu yapıştırın:
```bash
curl -sL https://githubusercontent.com | bash
```

### ⌨️ Kısayol Ayarları
Kurulum bittikten sonra sistem ayarlarınızdan şu kısayolları atamanız önerilir:
- **Başlat:** `sesle-yaz`
- **Durdur:** `nerd-dictation end`

---

## 🇺🇸 English Description

A stable and fast speech-to-text solution for Linux (specifically optimized for CachyOS and Arch-based distros). It works system-wide in any application.

### 🚀 Quick Installation
Open your terminal and run:
```bash
curl -sL https://githubusercontent.com | bash
```

### ⌨️ Shortcut Settings
After installation, it is recommended to set these custom shortcuts:
- **Start:** `sesle-yaz`
- **Stop:** `nerd-dictation end`

---

## ✨ Özellikler / Features
- **Auto-Timeout:** 3 saniye sustuğunuzda otomatik kapanır. / Automatically stops after 3 seconds of silence.
- **Safety First:** `--clearmodifiers` sayesinde rastgele pencereler açılmasını engeller. / Prevents accidental system commands.
- **Offline:** Verileriniz buluta gitmez, yerel çalışır. / Processes your voice locally for privacy.
